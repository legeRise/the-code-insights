# The Code Insight

A small Markdown-first publication built with Astro.

## Write

- Add articles to `src/content/articles/`.
- Add project notes to `src/content/projects/`.
- Put images in `public/images/` and reference them as `/images/name.webp`.
- Set `draft: true` in an article's frontmatter to hide it from builds.

## Run

```sh
npm install
npm run dev
```

Before deployment, replace `https://example.com` in `astro.config.mjs` and `public/robots.txt` with your real domain.
